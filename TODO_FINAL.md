# قائمة المهام النهائية - منصة بذره

## 1. توحيد الشروط والأحكام ونظام التفاوض وحماية الملكية
- [ ] فتح جميع صفحات الشروط والأحكام والخصوصية والأسئلة الشائعة ونموذج العمل
- [ ] تعديل النصوص لتعكس نموذج بذره الحقيقي
- [ ] توضيح أن بذره منصة وساطة ذكية بدون قروض أو حصص أو ملكية
- [ ] شرح نظام الدعم عبر باقات فقط
- [ ] توضيح أن جميع المستخدمين يمكنهم أن يكونوا (أصحاب مشاريع + داعمين + مسوقين + مستثمرين)
- [ ] شرح نظام "فك باب التفاوض" بالتفصيل
- [ ] توضيح أن "فك باب التفاوض" يفتح قناة خاصة داخل المنصة
- [ ] شرح باقة المستثمر ومزاياها
- [ ] كتابة نظام حماية الملكية الفكرية
- [ ] التأكد من عدم وجود تضارب بين النصوص

## 2. إزالة المعلومات الوهمية
- [ ] حذف جمل "انضم الآلاف" أو "آلاف الرواد"
- [ ] استخدام نصوص عامة بدون أرقام
- [ ] التأكد من أن أي رقم متبقي حقيقي

## 3. إصلاح التسجيل وتسجيل الدخول (بدون Manus)
- [ ] إلغاء الاعتماد على Manus OAuth
- [ ] نموذج التسجيل: اسم + بريد + كلمة مرور + يوزر (بدون نوع حساب)
- [ ] تفعيل نظام تفعيل البريد
- [ ] التأكد أن الحساب لا يعمل قبل التفعيل

## 4. التسجيل المبكر ونظام أكواد الإحالة
- [ ] إضافة إشارة "التسجيل المبكر" في نموذج التسجيل
- [ ] إضافة Popup يوضح فوائد التسجيل المبكر
- [ ] شرح نظام الإحالات والمكافآت
- [ ] ربط نظام الإحالة بقاعدة البيانات
- [ ] نصوص واضحة بدون وعود غير صحيحة

## 5. إصلاح اللوجو في الفوتر
- [ ] إصلاح شكل اللوجو (بدون تمدد أو قص)
- [ ] مطابقة خلفية اللوجو مع خلفية الفوتر
- [ ] التأكد من تناسق اللوجو في الهيدر والفوتر

## 6. تفعيل وكيل تقييم الأفكار
- [ ] التأكد أن صفحة التقييم تعمل بالكامل
- [ ] الوكيل يستلم الفكرة ويحللها ويرجع تقريرًا
- [ ] عرض النتائج بدون أخطاء

## 7. المجتمعات ولوحات الصدارة
- [ ] المستخدم يمكنه نشر بوست
- [ ] التفاعل والمتابعة تعمل
- [ ] البيانات تحفظ في القاعدة وتظهر بعد التحديث
- [ ] لوحات الصدارة تعرض بيانات حقيقية فقط

## 8. فحص روابط الهيدر والفوتر
- [ ] تجربة كل رابط في الهيدر
- [ ] تجربة كل رابط في الفوتر
- [ ] التأكد أن كل رابط يذهب لصفحة صحيحة

## 9. ربط كل شيء بقاعدة البيانات
- [ ] التسجيل يحفظ في القاعدة
- [ ] الدعم يحفظ في القاعدة
- [ ] البوستات تحفظ في القاعدة
- [ ] المشاريع تحفظ في القاعدة
- [ ] التقييمات تحفظ في القاعدة
- [ ] الإحالات تحفظ في القاعدة
- [ ] التعديلات تحفظ في القاعدة
- [ ] حذف أي بيانات ثابتة أو وهمية

## 10. لوحة الإدارة
- [ ] ربط حساب الإدارة ببريد info@bithrahapp.com
- [ ] إعطاء كامل الصلاحيات للحساب
- [ ] التأكد أن بقية المستخدمين ليس لهم صلاحيات

## 11. اختبار تجربة مستخدم كاملة
- [ ] تسجيل حساب جديد
- [ ] استقبال رسالة التفعيل
- [ ] تفعيل الحساب
- [ ] تسجيل الدخول
- [ ] تعديل البروفايل
- [ ] إنشاء مشروع كامل
- [ ] إضافة باقات دعم
- [ ] التأكد من وجود زر "فك باب التفاوض"
- [ ] الضغط على "فك باب التفاوض" والتأكد من فتح القناة
- [ ] نشر بوست في المجتمع
- [ ] تقييم فكرة
- [ ] تجربة رابط إحالة
- [ ] التأكد من عدم وجود Errors

## 12. التحقق النهائي
- [ ] "فك باب التفاوض" موجود داخل صفحة المشروع
- [ ] النظام موضح في الشروط والأحكام
- [ ] لا توجد حصص أو نسب ملكية
- [ ] دعم + باقات دعم + تفاوض واضحة
- [ ] التسجيل + تسجيل الدخول + التفعيل شغال
- [ ] التسجيل المبكر + الإحالات شغالة
- [ ] تقييم الأفكار شغال
- [ ] المجتمع ولوحات الصدارة شغالة
- [ ] الروابط كلها صحيحة
- [ ] كل شيء مرتبط بقاعدة البيانات
- [ ] لوحة الإدارة تعمل بحساب info@bithrahapp.com
