## Data Structure

### Stats Cards
```typescript
{
  totalIdeas: number;
  excellentIdeas: number; // score >= 80
  goodIdeas: number; // score 60-79
  averageScore: number;
}
```

### Filters
```typescript
{
  sectors: string[]; // ["ØªÙ‚Ù†ÙŠØ©", "ØµØ­Ø©", "ØªØ¹Ù„ÙŠÙ…", "ØªØ¬Ø§Ø±Ø©", ...]
  scoreRange: { min: number; max: number }; // 0-100
  stages: string[]; // ["Ù…Ø¬Ø±Ø¯ ÙÙƒØ±Ø©", "Ù†Ù…ÙˆØ°Ø¬ Ø£ÙˆÙ„ÙŠ", "Ù…Ù†ØªØ¬ Ø¬Ø§Ù‡Ø²"]
  budgetRange: { min: number; max: number }; // 0-1000000
  search: string;
  sortBy: "newest" | "highest_score" | "lowest_budget";
  page: number;
  pageSize: number;
}
```

### Idea Card Data
```typescript
{
  id: number;
  ideaName: string;
  ideaDescription: string; // truncated to 150 chars
  sector: string;
  category: string;
  stage: string;
  financialNeeds: string;
  overallScore: number;
  strengths: string[]; // top 3
  createdAt: Date;
  evaluatedAt: Date;
}
```

## Color Coding

- **Excellent (80-100)**: Green badge, green progress bar
- **Good (60-79)**: Blue badge, blue progress bar
- **Fair (40-59)**: Yellow badge, yellow progress bar
- **Poor (0-39)**: Red badge, red progress bar

## Features

1. **Advanced Filters**
   - Multi-select sectors
   - Score range slider
   - Stage checkboxes
   - Budget range slider
   - Reset all filters button

2. **Search & Sort**
   - Full-text search (name + description)
   - Sort by: newest, highest score, lowest budget
   - Results count display

3. **Idea Cards**
   - Score badge with color coding
   - Truncated description
   - Key metadata (sector, budget, date)
   - Quick view button
   - View details link

4. **Quick View Modal**
   - Full evaluation summary
   - All 5 dimension scores
   - Top 3 strengths
   - Top 3 weaknesses
   - Recommendations preview
   - Link to full details page

5. **Pagination**
   - 12 ideas per page
   - Page number buttons
   - Previous/Next navigation
   - Total results count
